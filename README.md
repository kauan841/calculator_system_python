# 🧮 Calculator System Python

Uma calculadora desenvolvida em Python com foco no aprendizado de lógica de programação, modularização, tratamento de exceções e boas práticas de desenvolvimento.

---

## 🚀 Sobre o Projeto

Este projeto foi criado para praticar conceitos fundamentais do desenvolvimento Back-End utilizando Python.

A aplicação funciona através do terminal e permite realizar diversas operações matemáticas de forma organizada e modular, utilizando funções separadas para cada operação.

---

## ✨ Funcionalidades Atuais

✅ Soma

✅ Subtração

✅ Multiplicação

✅ Divisão

✅ Potenciação

✅ Raiz Quadrada

✅ Menu interativo

✅ Tratamento de erros com `try` e `except`

✅ Código modularizado

✅ Funções separadas em arquivos

✅ Ambiente virtual (`venv`)

## 📂 Estrutura do Projeto

```text
calculator_system_python/
│
├── funcoes/
│   ├── soma.py
│   ├── subtracao.py
│   ├── multiplicacao.py
│   ├── dividir.py
│   ├── potencia.py
│   └── raiz.py
│
├── venv/
├── main.py
├── README.md
├── .gitignore
└── requirements.txt
```

> Observação: A pasta `venv` é utilizada apenas para o ambiente virtual local e não deve ser enviada para o GitHub.
